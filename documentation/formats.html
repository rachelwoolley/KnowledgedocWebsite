Documentation Formats

Knowledgedoc predominately uses single-sourced documentation software, Author-it, to produce our documentation. This software enables us to produce the same documentation across a number of formats.

Knowledgedoc can produce documentation in the format of:

  • printable formats such as Microsoft Word and PDF documents
  • online help formats that include links between topics and a search function
  • Web pages
  • context sensitive help for software applications
  • KdAssist, a mobile web application for tablets and smartphones.
