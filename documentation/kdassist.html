KdAssist

KdAssist is an available format for the documentation we produce. Developed by Knowledgedoc, KdAssist is a web application that can be easily used on a mobile, tablet or desktop. This is the perfect format for procedures that need to be accessed on site, no desktop computer or manual printout is required.

View the mobile and tablet version of the KdAssist demo. To view a demonstration of the responsive desktop and mobile version contact us.

Features

KdAssist features:

  • a search function
  • a history of all the topics you've viewed
  • the ability to bookmark topics
  • a table of contents and index
  • a see also section for related topics
  • topic feedback

KdAssist can be modified to suit your organisation's needs. Contact us to discuss potential additional features that you may require.
