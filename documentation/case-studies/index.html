Case Studies

In 2003, Synergetic Management Systems employed Knowledgedoc to write the documentation for their flagship software product, Synergetic Management System for Schools. Synergetic’s requirements were for Knowledgedoc to document the entire system and produce:

  • a PDF of each module for distribution to customers for printing
  • an online compiled help file with a table of contents, index and search facility
  • context sensitive help for each separate screen that displays when a user presses F1.

Synergetic had already started the documentation themselves using RoboHelp, but realised that they needed assistance to get the documentation professional-looking, complete and user-friendly.

The project took two writers six months to complete. Knowledgedoc has since been maintaining the documentation for each release of the software and several other software products produced by Synergetic Management Systems.
