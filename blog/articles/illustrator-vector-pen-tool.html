Creating a vector image in Illustrator - Using the Pen tool27/10/15

Welcome to the final part of the three-part series on creating a logo from a source file in Adobe Illustrator. In this last part we created most of the text in the logo. Now we have some text and a background. In this part we will use the Pen tool to create the final letter and finalise the logo.

To create the letter “E” using the Pen tool:

  1. Open the file from Part 2 in Illustrator
  2. Click .
    The Layers panel is displayed.
  3. Click to add a new layer.
  4. Double-click the new layer.
    The Layer Options window is displayed.
  5. Rename the layer "Letter E".
  6. Select the Pen tool.
  7. Click a corner of the white rectangle inside the “E” shape.
  8. Repeat step 3 to create a rectangle slightly longer than the white rectangle in the source image.
  9. Click the Selection tool.
  10. Click the rectangle while holding down the Alt key (Option key on a Mac).
    The mouse cursor changes to .
  11. Drag away from the rectangle.
    The rectangle is copied.
  12. Select both rectangles and move them off the source image.
  13. Select the Pen tool.
  14. Click the corner of the “E”.
  15. Repeat step 14 for the remaining corners to create a diamond.
  16. Drag the white rectangles over the diamond until they align with the rectangles in the source image.
  17. Select the diamond.
  18. Select Object > Arrange > Send to back.
  19. Shift-click to select the rectangles and the diamond.
  20. Select Window > Pathfinder.
    The Pathfinder panel is displayed.
  21. Click to minus the rectangles from the image diamond.
  22. Select Window > Align.
    The Align panel is displayed.
  23. Click to align all the objects along their bottom anchor point.
  24. The letter E is created.
  25. Save the image.

You should save the file as:

  • Adobe Illustrator (.ai) file
  • Either
    • EPS file (page layout, word processing and graphics)
    • SVG file (web content).

All the components of the vector image are complete. For a nice finishing touch you could merge the first three letters of the company name into a single shape using the Pathfinder panel.
