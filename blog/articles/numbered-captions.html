Numbered figure and table captions26/05/15

Last month I discussed how to create numbered headings to divide a Microsoft Word document into chapters. This week I’ll discuss how to apply table and figure caption numbering that is consistent with your heading numbers. For example your figure captions could be numbered 1-1, 1-2 and 1-3 in chapter 1 and 2-1, 2-2 and 2-3 in chapter 2.

To set up automatically numbered captions you need to use Word’s built in captions. The overall process for setting up numbered captions is:

  • open the Caption window for your particular table or figure
  • type in the caption for that object
  • set the label and position of the caption
  • add numbering to the caption.

To open the Caption window:

  • For a table:
    1. Hover your mouse over the table.
    2. Right click the icon in the top left corner.
    3. Select Insert Caption...
      The Caption window is displayed.
  • For a figure:
    1. Right click on the figure.
    2. Select Insert Caption...
      The Caption window is displayed.
  • From the References tab:
    1. Select the item you want to make a caption for.
    2. Open theReferences tab.
    3. Click Insert Caption in the Captions area.
      The Caption window is displayed.

To add chapter numbering:

  1. Open the Caption window.
  2. Click Numbering...
  3. Select the Include chapter number checkbox to number the captions 1-1, 1-2, 2-1, 2-2 etc.
  4. To change the numbering to reflect a different heading level change the value in the Chapter starts with style field. For example if you changed this value to Heading 2 your numbering system could be 1.1-1, 1.1-2, 1.2-1, 1.2-2 etc.
